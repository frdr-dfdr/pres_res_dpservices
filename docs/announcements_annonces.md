# Release of FRDR Format Policy Registry / Lancement du Registre des politiques relatives aux formats du DFDR

**Bilingual Message / Message bilingue**

The Federated Research Data Repository (FRDR) is pleased to announce the launch of its Format Policy Registry (FPR). The FPR is the outcome of years of file format work undertaken by the FRDR Team. The resource provides a listing of identified file formats in FRDR and an outline of the preservation actions for each. The goal of the FPR is to provide an overview of the digital preservation work that occurs behind the scenes.

The FPR comprises a strategy that outlines different preservation activities and is divided into 18 content type groups. Each group includes a listing of formats that belong to each content type. Every entry contains the file extension(s) associated with the format, the preservation action undertaken for a specific format, if one is available, and the level of preservation assigned to that format. The total number of files in FRDR that belong to a specific format is also provided.

This resource is not meant to be static and therefore will be updated when new preservation capabilities are available to FRDR or when new file formats are identified.

We hope that the RDM community will find the resource helpful and that it can function as a starting point for digital preservation activities at other repositories.

The FRP and accompanying resources can be found at https://github.com/frdr-dfdr/pres_res_fpr/. 

To get in contact with us please email preservation@frdr-dfdr.ca. 

The FRDR Team

Le Dépôt fédéré de données de recherche (DFDR) est heureux d’annoncer le lancement de son Registre des politiques relatives aux formats (FPR). Le FPR est le fruit de plusieurs années de travaux sur les formats de fichiers menés par l’équipe du DFDR. Cette ressource présente une liste des formats de fichiers identifiés dans le DFDR ainsi qu’un aperçu des actions de préservation associées à chacun d’eux. L’objectif du FPR est de donner un aperçu des activités de préservation numérique qui se déroulent en arrière-plan.
 
Le FPR s’appuie sur une stratégie définissant diverses activités de préservation. Il est structuré en 18 groupes de types de contenus, chacun présentant les formats associés à un type de contenu précis. Chaque groupe comprend une liste de formats associés à un type de contenu donné. Chaque entrée comprend les extensions de fichiers associées au format, l’action de préservation appliquée à ce format, le cas échéant, ainsi que le niveau de préservation qui lui est attribué. Le nombre total de fichiers présents dans le DFDR pour chaque format est également indiqué.
 
Cette ressource n’a pas vocation à être statique et sera donc mise à jour à mesure que de nouvelles capacités de préservation seront offertes par le DFDR ou lorsque de nouveaux formats de fichiers seront recensés.
 
Nous espérons qu’elle sera utile à la communauté de la gestion des données de recherche et qu’elle pourra constituer un point d’ancrage pour des activités de préservation numérique menées dans d’autres dépôts.
 
Le FPR et les ressources connexes sont accessibles à l’adresse suivante : https://github.com/frdr-dfdr/pres_res_fpr/.
 
Pour communiquer avec nous, veuillez écrire à preservation@frdr-dfdr.ca.
 
L’équipe du DFDR
